# GDV Module 3 - Pac-Man Inspired Game

## Overzicht

In deze module gaan studenten in duo's werken aan een eigen game geïnspireerd op Pac-Man. Het doel is om de geleerde vaardigheden uit M1 en M2 toe te passen en uit te breiden met nieuwe technieken.

## Eindopdracht

Ontwikkel in tweetallen een **Pac-Man geïnspireerde game met een eigen twist**. Studenten bedenken, ontwerpen en bouwen hun eigen variant van het klassieke spel.

## Leerdoelen

Na afloop van deze module kan de student:

- Een grid-based level genereren met code
- Input handling en beweging implementeren
- Collision detection toepassen (colliders, triggers, raycasts)
- Een waypoint systeem bouwen voor AI beweging
- Animaties aanroepen en creëren in Unity
- Game states beheren met een Game Manager
- UI elementen implementeren (score, tijd, levens)
- Een Game Design Document opstellen

## Lesstructuur

Elke les duurt **2 uur** en is als volgt opgebouwd:

- **Docent demo & uitleg** (max. 1 uur)
- **Oefeningen** (1 uur)

## Werkwijze

- Werk in **duo's** (2-tallen)
- Gebruik **Git** voor versiebeheer
- Documenteer je concept in een **Game Design Document**
- Test regelmatig met **playtests**

## Benodigdheden

- Unity (LTS versie)
- Visual Studio / VS Code
- Git & GitHub account
- Tekenprogramma voor concept art (optioneel)

## Beoordeling

De eindopdracht wordt beoordeeld op:

- Technische implementatie
- Originaliteit van de twist
- Game Design Document
- Samenwerking (Git history)
- Presentatie/demo
