# Les 7.2 — Reflectie & Voorbereiding Eindpresentatie

## Leerdoel

Na deze les kun je reflecteren op je leerproces en een eindpresentatie voorbereiden.

---

## Theorie

### Waarom Reflecteren?

Reflectie helpt je om:

- Te begrijpen wat je geleerd hebt
- Patronen in je werk te herkennen
- Beter te worden in toekomstige projecten

### Reflectie Methode: STARR

| Letter        | Vraag                  | Voorbeeld                                                            |
| ------------- | ---------------------- | -------------------------------------------------------------------- |
| **S**ituatie  | Wat was de context?    | "Bij les 3 moest ik collision detection implementeren"               |
| **T**aak      | Wat moest je doen?     | "Ik moest dots kunnen verzamelen"                                    |
| **A**ctie     | Wat deed je?           | "Ik zocht tutorials en probeerde verschillende approaches"           |
| **R**esultaat | Wat was het resultaat? | "Het werkte, maar ik begreep eerst niet waarom triggers nodig waren" |
| **R**eflectie | Wat heb je geleerd?    | "Ik begrijp nu het verschil tussen Collision en Trigger"             |

### Eindpresentatie

Je presentatie bevat:

1. **Inspiratie** — Waar komt je idee vandaan?
2. **Ontwerp** — Wat was het plan?
3. **Demo** — Laat je game zien (GIF of live)
4. **Wat ging goed** — Successen + lessen
5. **Wat was lastig** — Uitdagingen + lessen
6. **Trots** — Hoogtepunt

### Tips voor een Goede Presentatie

| Do                     | Don't                         |
| ---------------------- | ----------------------------- |
| Korte zinnen           | Hele paragrafen op slides     |
| Beelden en demo's      | Alleen tekst                  |
| Eerlijk over problemen | Doen alsof alles perfect ging |
| Concrete voorbeelden   | Vage uitspraken               |

## Toepassing

### Deze week:

1. **GIF maken:**
   - Download [ScreenToGif](https://www.screentogif.com/)
   - Neem 10 seconden gameplay op
   - Sla op als GIF

2. **Presentatie maken:**
   - Gebruik PowerPoint, Google Slides, of Figma
   - Max 7 slides
   - Meer beelden, minder tekst

3. **Oefenen:**
   - Presentatie duurt max 5 minuten
   - Oefen hardop (tijd jezelf!)
   - Vraag feedback van duo-partner

### Checklist Eindpresentatie

- [ ] Game naam en team duidelijk
- [ ] Inspiratie met afbeelding
- [ ] Ontwerp uitgelegd
- [ ] GIF van gameplay (max 10 sec)
- [ ] 2+ punten wat goed ging
- [ ] 2+ punten wat lastig was
- [ ] Duidelijk waar je trots op bent
- [ ] Geoefend en timing gecheckt
