## Hoe Deze Lessen Gebruiken

1. **Lees de Concept Uitleg** - Begrijp wat het concept doet en hoe het werkt
2. **Bestudeer de Praktijk Voorbeelden** - Zie hoe het in echte games wordt gebruikt
3. **Voltooi de Oefening** - Bouw een kleine project met het concept
4. **Probeer de Uitdagingen** - Verdiep jezelf met extra complexiteit

Elke les is **onafhankelijk** - je kan ze in willekeurige volgorde doen!

## Tips voor Succes

- **\*Type de code zelf** - Copy-paste helpt niet bij leren
- **Experimenteer** - Verander waarden en kijk wat gebeurt
- **Debug** - Gebruik Debug.Log() om te zien wat je code doet
- **Bouw voort** - Combineer concepten van verschillende lessen
- **Vraag om hulp** - Lukt iets niet? Vraag je docent!
