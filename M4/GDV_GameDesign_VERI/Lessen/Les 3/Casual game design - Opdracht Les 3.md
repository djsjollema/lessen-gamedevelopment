# Opdracht week 3 - Core Mechanic Afmaken

Deze week werk je verder aan je core mechanic. Aan het einde van de les heb je een speelbare versie die je kunt laten zien.

Je voegt geen nieuwe mechanic toe. Je maakt de bestaande mechanic af en lost problemen op.

## Voorwaarden

Je mechanic:

- bevat input, actie, gevolg en feedback
- is speelbaar in Unity
- bevat geen geweld
- past bij je gekozen uitdaging
- staat in je bestaande 2D Unity-project
- is gepusht naar GitHub

## Stap 1 - Core Mechanic Afmaken

Werk 25 minuten geconcentreerd aan je mechanic.

Focus op:

- ontbrekende onderdelen bouwen
- fouten oplossen
- variabelen goed instellen
- testen of de mechanic duidelijk werkt
- zorgen dat de mechanic speelbaar is

## Klaar?

Ga dieper, niet breder:

- test je mechanic meerdere keren
- verbeter de timing of besturing
- stel belangrijke variabelen beter af
- verbeter bestaande feedback
- los bugs op

Voeg geen nieuwe mechanic toe.
