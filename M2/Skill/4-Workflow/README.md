# Skill 4-Workflow

Dit project bevat een eenvoudige webpagina voor de workflow-les.

## Live demo

Bekijk de pagina via [index.html](https://djsjollema.github.io/lessen-gamedevelopment/M2/Skill/4-Workflow/).

## Gebruik op GitHub Pages

Om deze pagina live te tonen via GitHub Pages:

1. Push deze repository naar GitHub.
2. Ga naar de repository-instellingen op GitHub.
3. Zoek naar het onderdeel "Pages".
4. Selecteer de branch `main` en de root (`/`) als bron.
5. Sla op. Je krijgt een link naar de live-versie van de site.

De indexpagina zal dan bereikbaar zijn via een link zoals:
`https://<gebruikersnaam>.github.io/<repository-naam>/index.html`

Vervang `<gebruikersnaam>` en `<repository-naam>` door jouw GitHub gegevens.
