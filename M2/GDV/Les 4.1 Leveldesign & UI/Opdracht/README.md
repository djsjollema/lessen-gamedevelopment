# Les 4.1 Week 4 – Level design & UI Opdracht 

## Doel van deze week
Je maakt een nieuw level voor jouw Peggle-spel.  
Je bouwt een UI met een score en een eigen font.  
Je voegt minimaal één extra UI-element toe dat nuttig is voor jouw game.  
Je gebruikt anchors, Canvas Scaler en minstens één 9-slice element.  
Je maakt een gif waarin je laat zien hoe jouw UI schaalt in verschillende resoluties.

---

## Wat lever je in aan het einde van Week 4

### Level design
• Een nieuw level in jouw scene  
• Pegs bewust geplaatst (structuur, uitdaging, flow)  
• Level is speelbaar  

### UI
• Score zichtbaar in de UI  
• Een geïmporteerd font dat je gebruikt in jouw UI  
• Minimaal één extra UI-element, bijvoorbeeld:  
  titel, startknop, retryknop, leveltitel, overlay, ballen-teller, hit-teller  
• Minstens één UI-element dat 9-slice gebruikt  
• Anchors correct ingesteld  
• Canvas Scaler op Scale with Screen Size  

### README
• Screenshot van jouw level (editor view)  
• Gif van jouw UI in-game met twee resoluties  
• Korte uitleg van jouw levelkeuzes en UI-keuzes  
• Reflectie voor Week 4  

### Version control
• Minimaal één commit en één push deze week

## Bonus (optioneel)

Wil je verder gaan deze week? Maak dan één of meer van deze extra UI-features:

• Een startscherm voordat het spel begint  
• Een startknop die het spel laat starten  
• Een game over-scherm dat verschijnt wanneer je verliest  
• Een retryknop om opnieuw te beginnen

• Pegs logisch geplaatst

• Minimaal één extra UI-element toegevoegd

• Canvas Scaler staat goed
