## Stap 3 - Het genereren van een TextMesh Pro SDF Font Asset.

Omdat we voor onze UI een font willen dat op alle schermresoluties scherp en schaalbaar blijft, gebruiken we een SDF Font Asset. SDF betekent Signed Distance Field en zorgt ervoor dat tekst er altijd strak uitziet, ongeacht de grootte of zoomfactor.

- Je kunt dit doen door rechts te klikken op de font.
- Dan verschijnt het menu, je kies: create -> Text Mesh pro -> Font asset -> SDF.
<img width="1506" height="967" alt="image" src="https://github.com/user-attachments/assets/74296702-38b4-4c31-b0c1-0f71e2f12f7a" />

Het kan zijn dat TextMesh Pro er bij jou nog niet bij staat.Unity installeert TextMesh Pro niet standaard, maar het staat wel klaar in de Package Manager. Zodra jij voor het eerst een TMP component toevoegt (bijvoorbeeld TextMeshPro – Text), vraagt Unity of je de TMP Essentials wilt importeren. Daarna staat TMP volledig in je project.
