# Les: Rekenen met vectoren

## Lesdoelen

-   Vectoren bij elkaar optellen\
-   Een vector vermenigvuldigen met een scalair\
-   Een combinatie maken van beide bewerkingen\
-   Grafisch en rekenkundig werken met vectoren

------------------------------------------------------------------------

## Startactiviteit

**Vraag:**\
Je loopt 3 meter naar rechts en daarna 2 meter omhoog. Waar eindig je?

Laat leerlingen dit tekenen op ruitjespapier.

**Conclusie:**\
Bewegingen kun je samenvoegen tot één beweging: een vector.

------------------------------------------------------------------------

## Wat is een vector?

Een vector heeft:

-   een richting\
-   een lengte

Voorbeeld:

(3, 2)

Betekenis:

-   3 naar rechts\
-   2 omhoog

------------------------------------------------------------------------

## Vectoren optellen

### Regel

(a, b) + (c, d) = (a+c, b+d)

Dus:

-   x‑coördinaten optellen\
-   y‑coördinaten optellen

### Voorbeeld

(4,1) + (2,3) = (6,4)

### Grafisch

-   teken eerste vector\
-   zet tweede vector met beginpunt op eindpunt eerste\
-   resultaat = vector van begin tot eind

### Oefeningen

1.  (1,5) + (3,2)\
2.  (-2,4) + (5,-1)

------------------------------------------------------------------------

## Vermenigvuldigen met een scalair

Een scalair is een gewoon getal.

### Regel

k · (a,b) = (ka, kb)

### Voorbeeld

3 · (2,1) = (6,3)

### Belangrijk

-   positief → zelfde richting\
-   negatief → omgekeerde richting

### Oefeningen

1.  2(3,4)\
2.  -1(5,-2)

------------------------------------------------------------------------

## Combinatie

### Voorbeeld

2(1,3) + (4,-1)

Stap 1: vermenigvuldigen\
= (2,6) + (4,-1)

Stap 2: optellen\
= (6,5)

### Oefening

3(2,-1) + 2(-1,4)

------------------------------------------------------------------------
